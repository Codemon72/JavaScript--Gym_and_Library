#### Pop up Boxes
All pop up boxes are methods on the window object:

`Window.alert()`
Displays an alert dialog.

`Window.confirm()`
Displays a dialog with a message that the user needs to respond to.

`Window.prompt()`
Returns the text entered by the user in a prompt dialog.
